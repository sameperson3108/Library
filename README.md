Рабочий вариант backend для библиотеки
Создаётся событие, имеющее уникальный id, в котором книга имеет 1 из 3 статусов - свободна, занята, либо просрочена. Добавлены проверки, которые выводят логические ошибки, при не соответствии статусов
Система имеет свои CRUD запросы, которые были проверены и использованием программы POSTMAN
База данных - POSTGRESQL
